## 💡 AEHM Model Ke Liye Kya Important Hai?

Tumhara **AEHM** (Adaptive Emotional Human Model) agar **code generation** pe focus karega toh:

| Priority | Language Types |
|----------|---------------|
| **Must** | Python, JavaScript/TypeScript, SQL, Bash, C/C++, Rust |
| **High** | Java, Go, Julia, Lean/Coq, YAML/JSON, Regex |
| **Medium** | HTML/CSS, Docker, Terraform, Prolog, GLSL/CUDA, Protobuf |
| **Advanced** | Verilog/Chisel, TLA+, LLVM IR, WASM, Triton, ONNX |
| **Niche** | COBOL, Ada, APL, Forth (legacy system understanding) |

This is a comprehensive list of 28 programming languages, scripting languages, and data formats required to build a robust dataset for the AEHM. Each language has a specific role in the AEHM, and each language/format has a priority level. The priority level indicates the importance of the language/format in the AEHM. The higher the priority level, the more important the language/format is.
